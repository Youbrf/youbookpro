Un plugin de réservation moderne, flexible et évolutif, conçu pour les professionnels : salons, thérapeutes, indépendants et PME. Gérez vos rendez-vous, clients, employés et services depuis une interface intuitive et centralisée.

== Description ==

**YouBookPro** est une solution complète de gestion de réservation, pensée pour s’adapter à tous types de structures professionnelles. Optimisé pour WordPress, le plugin met l'accent sur la performance, la simplicité d’utilisation et la personnalisation.

### Fonctionnalités principales :
- Réservations en ligne avec créneaux dynamiques
- Gestion des services, clients, employés et disponibilités
- Formulaire de réservation personnalisable (via blocs Gutenberg)
- Interface d'administration centralisée
- Notifications automatiques (e-mails)
- Vue calendrier pour les administrateurs et employés
- Intégration facile dans n’importe quel thème WordPress

### En cours de développement :
- Paiements en ligne (Stripe / PayPal)
- Intégration Google Calendar
- Tableau de bord analytique (KPI)
- Multilingue / compatible WPML
- Rappels par e-mail et SMS

== Installation ==

1. Téléversez le dossier `youbookpro` dans le répertoire `/wp-content/plugins/`.
2. Activez le plugin depuis le menu "Extensions" de WordPress.
3. Configurez vos services, horaires et employés via le menu **YouBookPro** dans l’administration.

== FAQ ==

= Est-ce compatible avec tous les thèmes WordPress ? =
Oui, tant que votre thème respecte les standards WordPress. Aucune dépendance avec un constructeur spécifique.

= Puis-je personnaliser le formulaire de réservation ? =
Oui. Le plugin propose un bloc Gutenberg et des hooks pour une personnalisation avancée.

= Le plugin est-il adapté pour une équipe multi-utilisateurs ? =
Absolument. Chaque employé peut disposer de ses propres horaires et services associés.
